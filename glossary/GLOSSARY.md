# Glossary (Preferred Terms)

## General
- Codex: 코덱스
- repository / repo: 리포지토리(리포)
- environment: 환경
- task: 태스크
- agent: 에이전트
- sandbox: 샌드박스
- pull request / PR: 풀 리퀘스트(PR)

## Technical style rules
- Keep API / CLI / class names unchanged (unless a well-known Korean term exists)
- If a term repeats, keep the Korean rendering consistent
